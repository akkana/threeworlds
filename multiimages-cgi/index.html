EarthView

About EarthView:

EarthView expects a series of images in a maps/ subdirectory, generated by daynightimage.py. In theory you could set that up as a CGI, but I haven't done that part.

The list of expected image filenames is defined in earthglobe-multiimages.js. Use spacebar to advance through the list of images.

Written by Akkana Peck. Part of the ThreeWorlds Three.js explorations.

Uses the Three.js library.
